Var: Variables declared with var are function-scoped. They are only visible within the function where they are declared.
Const: Variables declared with const are block-scoped. the value of a const variable cannot be reassigned once it is assigned but, their properties or elements can be modified.
Let: Variables declared with let are block-scoped. They are only visible within the block (enclosed by curly braces) where they are declared.Variables declared with let are hoisted but not initialized. You'll get a ReferenceError if you try to use them before the declaration.

Null:
1. Represents an intentional absence of a value.
2. Behaves like an object.
3. Often used to signify that a variable has no value.
4. If we try to access properties or elements of null, it won't throw an error.
Undefined:
1. Represents the absence of an initialized value.
2. It is a data type in JavaScript.
3. When a variable is declared but not assigned a value, it is undefined.
4. Trying to access data that is not present results in an undefined value.
